# study-munit-tdd

## Сочетания 

- **Ctrl+Shift+B** - запуск задачи сборки
- **Ctrl+Shift+U** - запуск задачи тестов

- **Ctrk + ;   A** - запустить все тесты
- **Ctrk + ;   L** - запустить последние тесты
- **Ctrk + ;   E** - запустить проваленные тесты
- **Ctrk + ;   S** - показать результаты тестов
